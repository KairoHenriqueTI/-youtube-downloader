# 📹 YouTube Downloader

Site simples para baixar vídeos e playlists do YouTube em MP4.

## 🚀 Como Usar

### 1. Instalar dependências:
```bash
pip install -r requirements.txt
```

### 2. Iniciar o servidor:
```bash
python app.py
```

### 3. Acessar no navegador:
```
http://localhost:5000
```

## 📋 Funcionalidades

✅ Baixar vídeos individuais do YouTube  
✅ Baixar playlists completas  
✅ Conversão automática para MP4  
✅ Interface simples e bonita  
✅ Lista dos últimos downloads  
✅ Vídeos salvos em `~/Downloads/youtube-videos`  

## 💡 Dicas

- Cole a URL do vídeo ou playlist do YouTube
- Marque a opção "É uma playlist completa?" se for baixar uma playlist
- Os vídeos ficam salvos em MP4 prontos para copiar pro pendrive!

## 🎯 Perfeito para:

- Baixar vídeos para assistir no carro
- Salvar músicas e podcasts
- Download de playlists completas
- Armazenar vídeos offline
